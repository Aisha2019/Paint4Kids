/// Notes For The Project [Paint For Kids]

    1- We Can Set A Special Font With This Function 
        => void SetFont(const int iSize, const unsigned short usStyle, const fontfamily ffFamily, const char* cpFontName = NULL);
    
    2- We Can Draw a Point With This Function 
        => void DrawPixel(const int iX, const int iY);
    
    
    3- We Can Draw A Sharp Edge Rectangle Or Smooth Edge Rectangle With This Function By Using The Last Two Parameter Or Not 
        => void DrawRectangle(const int iX1, const int iY1, const int iX2, const int iY2, const drawstyle dsStyle = FILLED, const int iWidth = 0, const int iHeight = 0);
    
    
    4- We Can Draw Polygon With This Function => void DrawPolygon(const int* ipX, const int* ipY, const int iVertices, const drawstyle dsStyle = FILLED);
    
    
    5- We Can Draw A Circle By This Function by using The center And The Raduis [I Want To Re Write It To Take Two Pionts On The Diameter]
        => void DrawCircle(const int iX, const int iY, const int iRadius, const drawstyle dsStyle = FILLED);
        
    
    6- We can Draw Ellipse By using This Function 
        => void DrawEllipse(const int iX1, const int iY1, const int iX2, const int iY2, const drawstyle dsStyle = FILLED);
        
        
    7- We Can Draw An Image By Using This Function And Change Its Width And height If It Isn't Transparent
        => void DrawImage(const image *imgThis, const int iX, const int iY, const int iWidth = -1, const int iHeight = -1);
        
        
    8- We Can Pick The Color With This Function 
        => color GetColor(const int iX, const int iY);
